# State 211 Web Addresses

Official public 211 web address for every US state and DC, researched
2026-07-20 via web search (Firecrawl backend). These are the front doors to
each region's 211 resource database — the foundation for `us211-api`'s
nationwide coverage.

> Note: the 5 US territories (PR, VI, GU, AS, MP) do **not** operate a
> US-style standalone 211 program; no public 211 site was found for them.
